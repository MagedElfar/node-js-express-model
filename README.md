## Description

It is a basic model for simple user rest-api system include "authentication & authorization"

API documentation using swagger:
ex: [http://localhost:5000/api-docs/](http://localhost:5000/api-docs/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# production mode
$ npm start
```

## Build the app

```bash
# build
$ npm run build

# production mode
$ npm start
```

## Test

```bash
# unit tests
$ npm run test
```

## Database Migration

```bash
# migration
$ npm run migration
```
